# Akalmatika - Feedback Log Template

Gunakan tabel atau format ini untuk mencatat hasil wawancara dari setiap orang yang telah melihat demo Akalmatika. Kumpulkan semua *log* di folder ini.

---

### Profil Responden
*   **Nama / Inisial:** 
*   **Peran (Guru/Siswa/Ortu/Partner/Dll):** 
*   **Tanggal Wawancara:** 

### Evaluasi Produk
*   **Hal yang Mereka Pahami (Sinyal Kuat):**
    *   *(Contoh: Paham bahwa ini bukan sekadar kuis, tapi melacak akar masalah dari konsep dasar)*
*   **Hal yang Membingungkan (Gesekan UX):**
    *   *(Contoh: Sempat bingung bedanya "Drill" dan "Mastery Check")*
*   **Fitur yang Dianggap Paling Bernilai:**
    *   *(Contoh: Heatmap miskonsepsi di Dashboard Guru sangat disukai)*
*   **Fitur yang Dianggap Kurang Penting:**
    *   *(Contoh: Progress bar di Mastery dianggap kurang kelihatan)*

### Penilaian Kuantitatif & Kualitatif
*   **Komentar Spontan (Quotes):**
    *   *"..."*
*   **Skor Ketertarikan (1 - 5):** `[ ]`
    *(1: Sangat tidak tertarik, 5: Ingin beli sekarang juga)*
*   **Apakah Bersedia Mencoba Lagi Jika Sudah Rilis?** (Ya / Tidak / Mungkin)

### Tindakan Lanjutan
*   **Follow-up Action:**
    *   *(Contoh: Masukkan ke daftar waiting list email, perbaiki label tombol X di Sprint berikutnya)*

---
*(Catatan: Anda dapat membuat duplikat file ini dengan format penamaan `YYYYMMDD-NamaResponden.md` untuk pengarsipan)*
